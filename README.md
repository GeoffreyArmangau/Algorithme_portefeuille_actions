# Projet 7 OpenClassrooms : Optimisation d'algorithmes

## Description
Ce programme vise à optimiser le choix d'investissements parmi une liste d'actions, en maximisant le profit tout en respectant un budget donné. Il propose deux méthodes :
- Brute force : recherche exhaustive de toutes les combinaisons possibles.
- Optimized : approche optimisée pour réduire le temps de calcul.

## Structure du projet
- `brute_force.py` : implémentation de l'algorithme de force brute.
- `Optimized.py` : implémentation de l'algorithme optimisé.
- `test.py` : tests des algorithmes.
- `dataset1_Python+P7.csv`, `dataset2_Python+P7.csv`, `Liste+d'actions+-+P7+Python+-+Feuille+1.csv` : jeux de données.
- `rapport.txt` : rapport du projet.

## Utilisation
1. Installez les dépendances (aucune bibliothèque externe requise).
2. Lancez le script souhaité :
   - Pour la force brute : `python brute_force.py`
   - Pour l'optimisation : `python Optimized.py`
3. Les résultats s'affichent dans la console.

## Tests
Utilisez `test.py` pour vérifier le bon fonctionnement des algorithmes.
